Create New Repository, e.g. ksbh.github.io
choose public or private
create repository 


-clone repository to local computer
git clone https://github.com/JasOlean/KSBHPortfolio.github.io.git

- after clone
cd KSBHPortfolio.github.io

"git status" (now no commits yet will appear)

open github.io folder in vs code
code your implementation in vs code

if want to push something to the github.io folder, first 
check "git status" (red color is a updated file, not pushed yet file) 

"git add filename or . for all" to push file
"git status" again to check ready (green color)

"git commit -m "first commit" " to commit to the repository

last step
"git push" to send to the git server
